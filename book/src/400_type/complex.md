## Strings

Strings are a complex type that are made of array of chars with null terminator '\0', and by default is utf8 encoded:
```
str[]
```

## Number
Number type is an abstraction of intiger and float type. It can be imaginary too.
```
num[]
```

## Pointer
```
ptr[]
```
## Error

err[]
